PeakRDL C-Header
================

PeakRDL CHeader is a python package which can be used to generate a register
abstraction layer C Header from a SystemRDL definition.

Installing
----------

Install from PyPi using pip:

.. code-block:: bash

   python3 -m pip install peakrdl-cheader
